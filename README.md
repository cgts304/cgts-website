# CGTS - Education Platform

Welcome to **CGTS**, a modern and responsive educational website designed to help students prepare for competitive exams and upgrade their skills.

## 🚀 Features
- Responsive Design for all screen sizes.
- Clean and interactive User Interface.
- Built using HTML5, CSS3, and JavaScript.

## 📂 Project Structure
- `index.html` - The core structure of the website.
- `style.css` - Custom styling and layout designs.
- `script.js` - Logic and interactive components.

## 🛠️ How to run locally
1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
